## 目录结构

- `仪器分析.md`
  - 主笔记正文，建议始终把它当作主要编辑对象。
- `第一、二章 绪论+光学导论课件.pdf`
  - 原始课件参考。
- `提取的文本内容.txt`
  - 从课件提取的文字，适合做核对、补全和改写。

## 推荐工作方式

### 1. 以 Markdown 为主

- 后续修改优先直接改 `仪器分析.md`。
- 如果需要导出 PDF，再使用根目录的 `run_md_xelatex.bat`。

### 2. 内容整理原则

- 当前笔记更偏“提纲化+术语整理”，适合继续补：
  - 概念定义
  - 分类对比
  - 公式与指标解释
  - 高频考点总结
- 如果参考课件补内容，优先做结构化重写，而不是粘贴原始提取文本。

### 3. 资料关系

- `第一、二章 绪论+光学导论课件.pdf` 是原始来源。
- `提取的文本内容.txt` 是中间参考材料，不建议把它当作最终笔记。
- 真正用于协作和长期维护的文件仍然应是 `仪器分析.md`。

## 导出方式

如需导出 PDF，可在工作区根目录运行：

```bat
D:\latex0223\run_md_xelatex.bat "D:\latex0223\仪器分析\仪器分析.md"
```

这会：

1. 先把 Markdown 转成 LaTeX
2. 再用 XeLaTeX 编译

通常会在当前目录生成：

- `仪器分析_xelatex.tex`
- `仪器分析_xelatex.pdf`

如果这些导出文件只是临时产物，不建议长期作为主编辑对象。
