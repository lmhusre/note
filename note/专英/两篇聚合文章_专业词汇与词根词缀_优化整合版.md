# 三、文章核心术语拆分

## 1. polymer / monomer / macromolecule

- **polymer** = poly-（多） + mer（单元）→ 聚合物
- **monomer** = mono-（单） + mer（单元）→ 单体
- **macromolecule** = macro-（大） + molecule（分子）
  → 大分子

## 2. polymerization / initiation / propagation / termination

- **polymerization** = polymer + -ization→ 聚合过程
- **initiation** = initi-（开始） + -ation→ 引发
- **propagation** = propag-（扩展） + -ation→ 增长
- **termination** = termin-（结束） + -ation
  → 终止

## 3. unsaturated / unpaired / active state

- **unsaturated** = un-（不） + saturated（饱和）→ 不饱和的
- **unpaired** = un-（未） + paired（已配对）→ 未成对的
- **active state**
  → 活性状态

## 4. olefinic / diolefin / vinyl

- **olefinic** = olefin + -ic→ 烯烃类的
- **diolefin** = di-（二） + olefin→ 二烯烃
- **vinyl** 建议整体记忆
  → 乙烯基

## 5. coupling / disproportionation / chain transfer

- **coupling** = couple + -ing→ 偶合
- **disproportionation** 建议整体记忆→ 歧化
- **transfer** = trans-（转移） + fer（带去）→ 转移
- **chain transfer to polymer**
  → 向聚合物链转移

## 6. initiator fragment / radical / ionic

- **initiator fragment**→ 引发剂裂解后的碎片
- **radical**→ 自由基，建议整体记忆
- **ionic** = ion + -ic
  → 离子的

## 7. 聚合物名称拆分

- **polyethylene** = poly + ethylene→ 聚乙烯
- **polystyrene** = poly + styrene→ 聚苯乙烯
- **polyvinyl chloride** = poly + vinyl + chloride→ 聚氯乙烯
- **poly(vinyl acetate)** = poly + vinyl + acetate→ 聚醋酸乙烯酯
- **poly(methyl methacrylate)** = poly + methyl + methacrylate→ 聚甲基丙烯酸甲酯
- **polyamide** = poly + amide→ 聚酰胺
- **polyester** = poly + ester→ 聚酯
- **polyether** = poly + ether→ 聚醚
- **polyimide** = poly + imide→ 聚酰亚胺
- **polyurethane** = poly + urethane
  → 聚氨酯

# 四、高频词族整理

## 1. 以 polymer 为核心

- polymer：聚合物
- polymeric：聚合物的
- polymerization：聚合
- polymer chain：聚合物链
- degree of polymerization：聚合度

## 2. 以 initiate 为核心

- initiate：开始，引发
- initiator：引发剂
- initiation：引发
- initiator fragment：引发剂碎片
- initiation step：引发步骤

## 3. 以 propagate / grow 为核心

- propagate：传播、延伸
- propagation：增长
- growth reaction：增长反应
- growing chain：增长链

## 4. 以 terminate 为核心

- terminate：终止
- termination：终止
- termination reaction：终止反应
- coupling：偶合终止
- disproportionation：歧化终止

## 5. 以 chain transfer / branch 为核心

- chain transfer：链转移
- chain transfer to polymer：向聚合物链转移
- branch：分支
- branching：支化

## 1. 两篇文章的逻辑压缩成一句话

**Unsaturated vinyl monomers are activated by initiators, then grow through propagation, and finally stop by termination; chain transfer may lead to branching.**

**不饱和乙烯基单体在引发剂作用下形成活性中心，随后通过链增长不断加成，最终经终止反应结束；若发生链转移，则可能产生支化。**

## 最后建议：怎么背最有效

### 第一步：先背词根词缀

优先背：**poly-, mono-, un-, trans-, mer, initi, propag, termin, -ation, -ization**。

### 第二步：再背机理词

把 **initiation / propagation / termination / coupling / disproportionation / chain transfer** 连成一条线背。
