# 仪器分析 - 第一、二章及光学导论笔记

## 第一章 绪论 Introduction

### 1. 分析化学定义

- 研究物质化学组成、结构、形态与含量的科学
- 发展和应用各种方法、仪器和策略，获取物质在空间和时间方面组成和性质的信息

### 2. 分析化学的发展

**三次重大变革：**

| 时期 | 第一次变革       | 第二次变革   | 第三次变革             |
| ---- | ---------------- | ------------ | ---------------------- |
| 时间 | 20世纪初至30年代 | 20世纪40年代 | 20世纪70年代末至今     |
| 技术 | 化学分析（经典） | 物理仪器分析 | 分析科学               |
| 特点 | 单一技术         | 多学科融合   | 高灵敏、高选择、智能化 |

### 3. 分析化学的分类

#### 按分析目的分类：

- **定性分析**：物质的化学组分有哪些？
- **定量分析**：确定样品中目标待测组分的含量
- **结构分析**：确定目标待测组分的结构

#### 按分析方法分类：

**化学分析（经典）**

- 根据化学反应及化学计量关系
- 方法：重量分析、滴定分析
- 特点：准确、仪器简单

**仪器分析（现代）**

- 根据物质的物理性质或物理化学性质
- 方法：光谱法、色谱法、电化学法等
- 特点：灵敏、仪器昂贵、无需预处理

### 4. 仪器分析的分类

```
仪器分析
├─ 光学分析
│  ├─ 光谱法（发射、吸收、荧光、散射）
│  └─ 非光谱法（折射、散射、干涉、衍射、旋光）
├─ 电化学分析（电导、电位、电解、库仑、极谱、伏安）
├─ 色谱分析法（气相、液相、离子、超临界流体、薄层、毛细管电泳）
├─ 质谱法
└─ 热分析法
```

### 5. 主要性能指标

#### 准确度 (accuracy)

- **定义**：测得浓度与真实浓度的接近程度
- **相对误差**：用相对误差衡量
  $$
  E_r = \frac{x_i - x_t}{x_t} \times 100\%
  $$
- **评价方法**：
  - 用标准物质评价
  - 与公认方法对照
  - 加标准回收验证

#### 精密度 (precision)

- **定义**：各测量结果间相互一致性程度，表征随机误差大小
- **评价指标**：相对标准偏差 RSD%
  $$
  RSD\% = \frac{s}{\bar{x}} \times 100\%
  $$
- **重要关系**：
  - 精密度高是准确度高的前提
  - 精密度高≠准确度高（受系统误差影响）
  - 准确度及精密度都高 → 结果可靠

#### 灵敏度 (sensitivity)

- **定义**：被测物浓度改变一个单位时所引起的响应信号的变化
- **公式**：$S = \frac{dR}{dC}$
- **应用**：校准曲线的斜率越大，灵敏度越高

#### 检出限 (LOD) 和定量限 (LOQ)

- **检出限**：定量分析实际可检测的最小量或最小浓度
  $$
  LOD = \frac{3s_b}{b}
  $$
- **定量限**：可以达到检测的最小量或最小浓度
  $$
  LOQ = \frac{10s_b}{b}
  $$
- 其中：$s_b$ 为空白信号的标准偏差，$b$ 为校准曲线的斜率

#### 线性范围

- 校准曲线保持线性或斜率保持常数的待测物浓度范围

### 6. 定量分析方法

#### 1. 校准曲线法（外标法）

- **原理**：用不同浓度的标准溶液配制校准曲线
- **要求**：
  - 被分析溶液和校准溶液的组成尽可能一致
  - 仪器操作参数一致
- **特点**：适用大多数方法，制作简单，无需预处理

#### 2. 内标法

- **原理**：加入固定量的内标物，获得被测物与内标物的信号比
- **特点**：消除因仪器波动/操作误差（进样量、检测器漂移等）
- **内标物要求**：
  - 物理和化学性质与被分析物相似
  - 不干扰被分析物的响应
  - 含量应在同一数量级以内

#### 3. 多点标准加入法（加标法）

- **原理**：将不同量的标准品加入样品中（在同一基质背景下）
- **特点**：消除基体效应的影响
- **操作步骤**：
  1. 取5份等体积样品
  2. 加入不同量标准液（0, 2, 4, 6, 8 μg等梯度）
  3. 定容至相同体积
  4. 测定并记录信号值
  5. 绘图：以加标量为x轴，信号为y轴
  6. 反推：直线与x轴交点的绝对值即为原始样品含量

## 第二章 光学分析法导论 Introduction to Optical Analysis

### 1. 光谱分析法

#### 定义

- **光谱**：复合光经分光后按波长或频率排列的图谱
- **光谱分析法**：测量物质光谱，根据光谱获取物质化学组成和结构信息的方法

#### 基础

- 能量作用于物质后产生电磁辐射信号
- 电磁辐射与物质相互作用后产生辐射信号变化
- 作用方式：发射、吸收、散射

#### 三个主要过程

1. 能源提供能量
2. 能量与被测物质相互作用
3. 产生被检测的信号

#### 三个基本特点

1. 所有光分析法均包含三个主要过程
2. 选择性测量，不涉及混合物分离
3. 涉及大量光学元器件

### 2. 电磁辐射与电磁波谱

#### 电磁辐射的波粒二象性

**波动性**

- 参数：波长(λ)、频率(ν)、波数(σ)
- 关系式：
  - $c = \nu \lambda$
  - $\sigma = \frac{1}{\lambda}$ (cm⁻¹)
  - $c = 2.998 \times 10^{10}$ cm/s

**粒子性**

- 普朗克公式：$E = h\nu = \frac{hc}{\lambda}$
- 其中：
  - E：光子能量 (J)
  - h：普朗克常数 = 6.626 × 10⁻³⁴ J·s
  - ν：频率 (Hz)
  - λ：波长 (cm)

#### 常用的电磁波谱

| 波长范围     | 典型光谱学 | 量子跃迁类型   |
| ------------ | ---------- | -------------- |
| 200-400 nm   | 近紫外     | 价电子         |
| 400-750 nm   | 可见光     | 价电子         |
| 0.75-2.5 μm | 近红外     | 分子转动和振动 |
| 2.5-50 μm   | 中红外     | 分子转动和振动 |
| 50-1000 μm  | 远红外     | 分子转动和振动 |
| 1-300 mm     | 微波       | 分子转动       |
| 0.3-1000 m   | 核磁共振   | 核自旋         |

### 3. 光谱分析法的分类

#### 按产生方式分类

- **发射光谱**
- **吸收光谱**
- **拉曼散射光谱**

#### 按形状分类

- **线光谱**（原子光谱）
  - 产生于气相单个原子或离子
  - 线宽约为 10⁻³ nm
- **带光谱**（分子光谱）
  - 产生于分子
  - 带宽达几个至几十个 nm
- **连续光谱**
  - 产生于固体
  - 通常产生背景干扰

#### 按分析对象分类

**原子光谱法**（线光谱）

- 基于原子外层电子跃迁
- 主要方法：
  - 原子发射光谱（AES）
  - 原子吸收光谱（AAS）
  - 原子荧光光谱（AFS）
  - X射线荧光光谱

**分子光谱法**（带光谱）

- 基于分子电子能级、振动-转动能级跃迁
- 主要方法：
  - 紫外-可见吸收光谱（UV-vis）
  - 红外光谱（IR）
  - 分子荧光光谱（MFS）
  - 核磁共振与顺磁共振波谱（NMR）

**非光谱法**

- 不涉及能级跃迁
- 仅改变传播方向等物理性质
- 方法：偏振、干涉、旋光法等

### 4. 原子光谱

#### 产生条件

1. 气态自由原子
2. 外层电子在不同能级间跃迁

#### 主要分析方法

**原子发射光谱分析法**

- 以火焰、电弧、等离子炬等作为光源
- 使气态原子外层电子受激而发射特征光谱
- 用于定性和定量分析

**原子吸收光谱分析法**

- 利用待测元素共振线的吸收
- 测定气态样品中原子对共振线的吸收程度
- 用于定量分析

**原子荧光光谱分析法**

- 测量原子蒸气在特定辐射激发下的荧光发射强度
- 用于测物质含量

> 原子发射光谱法（AES）已单列为第三章，导论部分仅保留原子光谱基础概念。

#### 原子谱线的三个参数

**1. 谱线波长**

- 由能级差决定
- 公式：$\Delta E = E_j - E_i = h\nu = \frac{hc}{\lambda}$
- $\lambda = \frac{hc}{E_j - E_i}$

**2. 谱线强度**

- *发射光谱*：$I = A_{ji}h\nu n_j$
  - $n_j$：处于能态j的原子密度
  - $A_{ji}$：自发发射爱因斯坦系数
- *吸收光谱*：$I_a = B_{ij}U h\nu n_i$
  - $B_{ij}$：受激吸收跃迁几率
  - U：照射光的能量密度

**3. 谱线形状**

- 描述：谱线强度随波长或频率的分布曲线
- 特征参数：中心频率和半宽度
- 半宽度：谱线峰高一半处的宽度

#### 原子谱线变宽原因

**1. 自然变宽**

- 根据测不准原理
- 处于高能态原子的寿命越短，谱线越宽
- 数量级：10⁻⁵ nm

**2. 热变宽（多普勒变宽）**

- 与原子的不规则运动有关
- 与原子质量(M)和温度(T)相关
- 数量级：10⁻³ nm

**3. 碰撞变宽（压力变宽）**

- 原子与其它组分相互碰撞引起的变宽
  - 外来气变宽（洛仑兹变宽）
  - 共振变宽（赫尔兹马克变宽）
- 数量级：10⁻³ nm

**4. 自吸和自蚀**

- **自吸**：光源中心高温原子的发射被周围低温同种基态原子吸收
- **自蚀**：自吸严重时，谱线中心的辐射被显著吸收

**5. 场致变宽**

- **斯塔克变宽(Stark Broadening)**：由电场效应引起
- **塞曼变宽(Zeeman Broadening)**：由磁场效应引起的谱线分裂

> 在1000～3000K、0.101MPa状态下，**热变宽和压力变宽是谱线变宽的主要因素**

### 5. 分子光谱

#### 分子能级系统

- **电子能级差**：最大 → 决定光谱带的位置
- **振动能级**：次之 → 在每个电子能级上叠加多个振动能级
- **转动能级**：最小 → 在每个振动能级上叠加极小的转动能级

#### 分子能级跃迁

$$
\Delta E = \Delta E_{电子} + \Delta E_{振动} + \Delta E_{转动} + \Delta E_{平动}
$$

#### 分子光谱的形状

**分子发射光谱**

- 半宽度：20~100 nm

**分子吸收光谱**

- 显示为带状结构

#### 分子光谱为带状光谱的原因

1. 分子对电磁辐射的吸收是分子能量变化的和
2. 溶液中相邻分子间的碰撞导致谱带加宽
3. 气相中的多普勒变宽和碰撞变宽超过转动谱线间的间距

#### 物质状态对谱带形状的影响

|       物质状态       |           结构特征           |     光谱类型     |
| :------------------: | :--------------------------: | :--------------: |
|       稀薄气体       |     振动、转动结构均存在     | 原子光谱（线状） |
|     中等气体压力     |  转动结构消失，振动结构存在  | 分子光谱（带状） |
| 分子间作用力小的液体 | 转动结构消失，振动结构不明显 |     精细结构     |
| 分子间作用力大的液体 |     转动和振动结构都消失     |     带状光谱     |

**基本单元**

- 光源
- 分光系统
- 进样系统
- 检测器
- 显示与数据处理

## 第三章 原子发射光谱法 Atomic Emission Spectroscopy (AES)

### 1. 基本原理与分析依据

- 气态原子/离子在热激发或电激发下，外层电子跃迁并发射特征线光谱。
- 定性分析依据：元素能级结构不同，故特征谱线波长不同。
- 定量分析依据：在一定条件下，谱线强度与浓度相关。

#### 补充：电离与离子线

- 当激发能足够大时发生电离，产生带正电离子与自由电子。
- 一次、二次电离离子也可发射谱线。
- 光谱表中常用罗马数字标记：
  - I：中性原子线；
  - II：一次电离离子线；
  - III：二次电离离子线。

### 2. 谱线强度关系与自吸修正

- 发射谱线强度常写作：$I_{ij}=N_iA_{ij}h\nu_{ij}$。
- 激发态粒子数服从玻尔兹曼分布，故强度受原子数、跃迁概率、统计权重、激发能、温度等影响。
- 在常用近似下，基态原子数可与浓度相关，得到定量近似 $I=aC$。
- 存在自吸时，常用经验关系：$I=aC^b$。
  - $b=1$：近似无自吸；
  - $b<1$：有自吸，$b$ 越小自吸越严重；
  - 自吸严重可出现自蚀（线中心被显著吸收）。

### 3. 仪器组成与分析流程

#### 核心组成

- 试样引入系统
- 激发光源
- 分光系统（单色器：入射狭缝、准直装置、色散装置、聚焦成像系统）
- 检测器与信号处理系统

#### 分析流程

1. 试样引入光源
2. 蒸发、解离、原子化
3. 原子/离子激发并发射
4. 分光与检测
5. 定性判别与定量计算

#### 常见进样方式

- 液体：气动雾化、超声雾化
- 固体：直接进样、电弧/火花熔融、电热蒸发、激光熔融等

#### 激发光源的基本要求

- 温度高，且应具有较好的稳定性与重现性。
- 背景尽量小，并兼顾操作简便与安全性。
- 光源中通常依次经历：试样熔融/蒸发、解离、原子化、激发或电离、发射以及部分自吸等过程。

#### 补充：单色器与色散方式

- 单色器的作用是把复色光按波长展开，便于选择分析线并降低背景干扰。
- 棱镜色散：利用不同波长光折射率不同而分光。
- 光栅色散：利用衍射和干涉联合作用分光，现代仪器中更常见。

#### 常见检测方式

- 目视法：适用于可见区，常借助看谱镜进行快速半定量判断。
- 摄谱法：用感光板记录谱线，依据谱线位置定性，也可依据黑度做半定量或定量。
- 光电法：利用光电转换器件将光信号转为电信号，是当前主流方式，便于自动化与高精度定量。

#### 常见检测器与直读仪器

- 光电倍增管（PMT）：由光阴极和多级倍增极组成，增益高，适合弱光检测。
- 光二极管阵列：属于多通道检测器，可同时接收不同波长信号。
- CCD检测器：可多谱线同步检测，响应快、体积小、稳定性好。
- 摄谱仪：以棱镜或光栅为色散元件、以感光板记录光谱，便于长期留存谱图。
- 光电直读光谱仪可分为多道直读、单道扫描和全谱直读三类。
- 多道直读与单道扫描仪器常配PMT，全谱直读仪器常配CCD。
- 全谱直读光谱仪的优点是光学元件固定、稳定性好、响应快，可同时检测多条谱线。

### 4. 激发光源类型与比较

#### 直流电弧

- 优点：蒸发能力强、灵敏度较高。
- 局限：稳定性较差，多用于定性或半定量。
- 适用：矿物、纯物质中痕量杂质的定性或半定量分析。

#### 低压交流电弧

- 优点：稳定性较好，可用于定量。
- 局限：有弧光漂移，对精度有影响。
- 注意：一般不宜分析低熔点金属。

#### 高压火花

- 优点：激发温度高、稳定性较好，适于固体分析。
- 局限：检出限相对较差，不利于痕量元素。
- 适用：高含量、难激发或低熔点元素分析。

#### ICP（电感耦合等离子体）

- 高频电磁场耦合产生高温等离子体，是当前最常用AES光源。
- 可实现高稳定、高灵敏、多元素测定。

#### 光源比较（记忆版）

| 光源         | 稳定性 | 蒸发/激发能力    | 典型应用场景         |
| ------------ | ------ | ---------------- | -------------------- |
| 直流电弧     | 较差   | 蒸发强、灵敏较高 | 固体定性/半定量      |
| 低压交流电弧 | 较好   | 综合较均衡       | 固体定量             |
| 高压火花     | 好     | 激发温度高       | 高含量或难激发元素   |
| ICP          | 很好   | 高温、高稳定     | 溶液多元素高精度定量 |

### 5. ICP-AES 重点（高频考点）

#### ICP组成与工作机制

- 常见等离子体光源包括直流等离子体、微波等离子体和ICP，其中ICP应用最广。
- 主要包括：高频发生器与感应线圈、三同心石英炬管、供气与载气系统。
- 高频发生器常见频率约 27-50 MHz，输出功率约 2-4 kW；感应线圈常为 2-5 匝水冷铜管。
- 外层石英管通冷却Ar，用于保护炬管并形成利于进样的中心低压通道。
- 中层气流用于维持等离子体；内层石英管由载气携带样品气溶胶进入等离子体。
- 常用Ar作工作气体，化学惰性好、光谱背景相对简单。
- 高频电流经线圈产生交变电磁场，点火后形成载流子并感生涡流加热，建立稳定等离子体。
- 样品气溶胶进入等离子体后在高温区原子化与激发，产生发射谱线。

#### 焰炬分区与测光区

- 焰心区：温度最高，连续背景强，通常不作为测光区。
- 内焰区：原子线/离子线强，是主要测光区（常以观测高度优化信号）。
- 尾焰区：温度较低，适于部分低激发能谱线。

#### ICP优势参数

- 检出限低（课件给出常见 $10^{-7}\sim10^{-9}\,g$ 量级）。
- 精密度较好（课件示例 RSD 常可小于 10%）。
- 线性范围宽（约 4-6 个数量级）。
- 可同时或顺序测定多元素，课件示例可同时测定约 56-72 种元素。
- 自吸/自蚀相对较小，基体效应相对较弱，准确度较好。

### 6. AES定性分析方法

#### 基本术语

- 分析线：用于判断某元素存在的特征谱线。
- 灵敏线：最易激发、最强的一条或几条谱线。
- 最后线：随浓度下降最后消失的谱线，常具有较高灵敏度。

#### 常用定性策略

1. 标准试样光谱比较法：与标准样在同条件下比对特征线。
2. 标准图谱比较法：以标准图谱（如铁谱）作波长标尺判别元素。

#### 判据

- 通常需至少2条（或更多）不受干扰的特征线共同确认元素存在。

### 7. AES定量分析方法

#### 基本关系

- 经验关系：$I=aC^b$ 或 $\,\lg I=b\lg C+\lg a$。

#### 内标法（常用）

- 用分析线与内标线强度比 $R=I_1/I_2$ 进行定量，可补偿光源波动与进样波动。
- 内标法选线原则：
  1. 内标元素与被测元素蒸发/激发行为相近。
  2. 分析线对宜同类型（原子线对原子线、离子线对离子线）。
  3. 激发电位相近、波长接近。
  4. 尽量无自吸、无重叠干扰。
  5. 内标元素含量恒定。

#### 标准曲线法

- 以标准样建立 $I$ 或 $R$ 与浓度关系，再回算样品浓度。

#### 标准加入法

- 用于基体匹配困难样品，可减小或消除基体效应造成的误差。

### 8. 工作条件优化与干扰控制

#### 条件优化要点

- 光源与激发参数（功率、电流、观测高度）
- 波长与狭缝宽度选择（分辨率与信噪比平衡）
- 样品前处理与进样稳定性
- 标准物质与校准策略选择

#### 抑制干扰常见手段

- 基体匹配与标准加入
- 使用光谱缓冲剂（改善稳定性与重现性）
- 使用光谱载体（稳定蒸发行为与弧温）
- 选用干扰更小的分析线/内标线对

### 9. 方法特点、局限与应用

#### 优点

- 多元素同时分析能力强
- 既可定性也可定量
- 灵敏度与选择性较好
- 在材料、环境、食品等领域应用广

#### 局限

- 对部分非金属元素（如卤素、稀有气体）适用性较差
- 部分光源条件下背景与光谱干扰仍需精细校正

#### 典型应用

- 金属与合金成分分析
- 土壤/水体/大气中的金属污染监测
- 食品中痕量有害元素检测
- 工业过程质量控制与在线监测

### 10. 拓展了解：AES前沿方向

- 微等离子体光源：装置趋于小型化、集成化，可在部分场景替代传统大型激发光源。
- 进样方式持续微型化：化学蒸气发生、光化学蒸气发生、固相微萃取等可与AES联用。
- 数据处理智能化：机器学习可用于复杂背景校正、特征提取和定量建模。

## 关键概念总结

### 对比区别

| 特征     | 原子光谱          | 分子光谱                 |
| -------- | ----------------- | ------------------------ |
| 产生源   | 气态自由原子      | 分子                     |
| 产生原因 | 外层电子能级跃迁  | 电子、振动、转动能级跃迁 |
| 光谱形状 | 线光谱            | 带光谱                   |
| 半宽度   | 10⁻³～10⁻⁴ nm | 20~100 nm                |
| 分析用途 | 元素分析          | 化合物结构分析           |

### 分析方法选择要点

- **样品类型**：单一组分/混合物
- **分析类型**：定性/定量
- **精度要求**：准确度/精密度
- **样品性质**：挥发性/热稳定性
- **成本考虑**：仪器成本/运行成本

### 学习要点

1. 掌握仪器分析方法的基本原理
2. 了解仪器的结构和工作原理
3. 理解分析步骤和数据处理方法
4. 学会根据样品性质选择合适的分析方法
5. 评估方法的适用条件和局限性
