## 推荐工作方式

### 1. 主文件优先

- 如果只是补正文、改公式、调结构，优先改 `分析化学.tex`。
- 不要直接手改 `auxil/` 里的 `.aux`、`.toc`、`.log`。

### 2. 扩充时的思路

- 先看现有 `\section` / `\subsection` 结构，优先在原框架下补内容。
- 对照课件文本时，优先提炼“定义、方法、公式、例题模板、易错点”，不要机械搬运整段原文。
- 如果补充统计处理、滴定分析等内容，尽量保持“概念 -> 公式 -> 解释 -> 例题/结论”的节奏。

### 3. 资源文件的使用方式

- `第二章分析化学中的误差及数据处理-课件.txt` 适合作为误差理论与统计处理部分的原始参考。
- `chapter3_extracted_clean.txt` 适合作为第三章或后续相关内容的扩写起点。
- `out/chapter4_acid_base_1.txt` 与 `out/chapter4_acid_base_2.txt` 可作为第四章复核、补例题与考前压缩版整理的文本依据。
- 如果未来继续新增提取文本，请在本 README 里补一句用途说明。
