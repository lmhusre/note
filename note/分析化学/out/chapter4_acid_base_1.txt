物料平衡：在一个化学平衡体系中，某一给定物质的

 总浓度（分析浓度）与各有关型体的平衡浓度之和相等。
 浓度为 c 的H3PO4溶液
                 —
 [H3PO4] + [H2PO4 ] + [HPO42—] + [PO43—] = c

  浓度为 c 的Na2SO3溶液

           [SO32—] + [HSO3—] + [H2SO3] = c

 溶液中正离子所带正电荷的总数等于负离子所带负
