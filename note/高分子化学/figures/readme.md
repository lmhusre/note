# 图片资源说明

本目录存放 `高分子化学.tex` 使用的图片资源。当前最重要的子目录是：

- `ch2_formula_only/`
  - 第二章“缩聚与逐步聚合”中使用的纯公式截图。

## 当前约定

- 图片应优先服务于正文，不应喧宾夺主。
- 只要正文已经有 LaTeX 公式，截图就只承担“保留课件原貌”和“辅助识别结构”的作用。
- 若图片与正文冲突，以课程 PDF 为准，修改图片或正文使两者重新一致。

## 命名规则

当前命名采用：

```text
<topic>_formula.png
```

例如：

- `epoxy_formula.png`
- `pc_formula.png`
- `pu_formula.png`

如果只是重裁旧图，请尽量沿用原文件名；如果是新增完全不同的图，再新增文件名。

## 裁图原则

- 只保留公式本体和必要的紧邻说明。
- 不保留页码、页眉、整页标题、无关段落。
- 若公式过长，可保留与公式直接相关的 `n` 范围、关键试剂、反应条件。
- 分辨率宁高勿低，优先保证打印和缩放后仍能看清。
- 如果一张图“看不全”和“看不清”同时存在，优先重新截图，不要单纯在 LaTeX 里硬放大。

## 推荐处理流程

### 1. 先定位引用

在 `高分子化学.tex` 中搜索：

- `includegraphics`
- 对应小节名
- 图片文件名

### 2. 判断改法

- 只是裁切不完整：直接替换原图文件。
- 只是放得太小：先判断是否能通过版式解决。
- 图文对应错误：优先查课程 PDF，必要时同时改图片和正文。

### 3. 替换后检查

- 编译 PDF。
- 检查图片是否完整。
- 检查说明文字和图片是否仍然一一对应。

## 不建议保留的临时文件

以下类型的文件如果只是中间步骤，交接前应删除：

- 临时渲染页
- 接触表图
- 一次性裁图脚本
- 仅用于目测定位的网格图

如果某个脚本确实会被反复复用，再考虑保留，并在这里补充用途说明。
