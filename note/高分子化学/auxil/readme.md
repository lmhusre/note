# `auxil` 目录说明

这个目录是编译中间文件目录，不是正文目录。

## 常见文件

- `高分子化学.aux`
  - 交叉引用中间文件。
- `高分子化学.toc`
  - 目录中间文件。
- `高分子化学.log`
  - XeLaTeX 编译日志。
- `高分子化学_build.log`
  - `build_tex_watch.py` 的构建日志。

## 使用方式

- 需要排查编译错误时，优先看这里的 `.log`。
- 如果目录、页码、交叉引用不稳定，通常需要重新编译 2 到 3 次。
- 这里的文件由构建过程自动生成，不要手动编辑。

## 与 `out/` 的区别

- `auxil/` 放中间文件和日志。
- `out/` 放最终生成的 PDF 和 `synctex` 文件。

## 协作提醒

- 交接时不必汇报 `.aux` 或 `.toc` 的差异，除非它们反映出正文结构真的有问题。
- 真正需要说明的是：是否编译成功、PDF 是否生成、日志里有没有新的报错。
