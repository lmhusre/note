# 专英笔记协作说明

这个目录存放的是专业英语学习笔记，当前主要围绕聚合相关英文材料进行整理。主维护格式是 Markdown。

## 当前文件组成

目前目录中有三份主要 Markdown 笔记：

- `Free_Radical_Vinyl_Polymerization_精编笔记.md`
  - 自由基乙烯基聚合的精编预习笔记。
- `UNIT2_Chain_Polymerization_中英对照全文.md`
  - 链式聚合材料的中英对照整理版。
- `两篇聚合文章_专业词汇与词根词缀_优化整合版.md`
  - 两篇聚合文章的专业词汇、词根词缀与高频术语整合版。

同时，每份 Markdown 当前都配有对应 PDF，便于阅读和分发。

## 这组笔记的特点

这不是单一文档项目，而是一组互相关联的学习材料：

- 一份偏“全文对照”
- 一份偏“精编理解”
- 一份偏“词汇与词根词缀整理”

因此协作时要先确认你改的是哪一类文档，不要把三种用途混在一起。

## 推荐维护方式

### 1. 先判断文档类型

- 如果目标是保留原文脉络，优先修改“中英对照全文”。
- 如果目标是提高复习效率，优先修改“精编笔记”。
- 如果目标是强化术语积累，优先修改“词汇与词根词缀整合版”。

### 2. 保持风格统一

- 中英对照文档：
  - 尽量保持“英文原句 + 中文解释/翻译”的稳定节奏。
- 精编笔记：
  - 优先保留“逻辑拆解、机制理解、一句话记忆、考点提示”的结构。
- 词汇整合文档：
  - 优先保留“词汇表、词根词缀、术语拆分、高频清单”的学习导向。

### 3. PDF 不是主编辑对象

- 目录中的 PDF 更适合作为阅读版或分发版。
- 真正应维护的是 `.md` 文件。

## 导出方式

如果需要把某份 Markdown 重新导出为 PDF，可在工作区根目录运行：

```bat
D:\latex0223\run_md_xelatex.bat "D:\latex0223\专英\<某个文件>.md"
```

例如：

```bat
D:\latex0223\run_md_xelatex.bat "D:\latex0223\专英\Free_Radical_Vinyl_Polymerization_精编笔记.md"
```

这会生成对应的 `_xelatex.tex` 和 `_xelatex.pdf`。

## 协作提醒

- 文件名较长时，修改前请先确认目标文件，避免改错文档。
- 如果新增新的专业英语笔记，建议继续沿用“主题 + 用途”的命名方式。
- 若两份文档之间存在术语引用关系，修改时尽量同步。

## 交接前检查

- 是否改的是正确那一份 Markdown
- 中英文对应关系是否仍然正确
- 术语、缩写和化学名词是否统一
- 如有导出 PDF，是否成功生成且没有把中间导出文件当作主文件
